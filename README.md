# Bluestock Mutual Fund Analytics Capstone

## Project Overview

This project analyzes Indian mutual fund industry data using Python, SQL, and Power BI. The objective is to build a complete analytics pipeline covering ETL, exploratory data analysis, performance analytics, risk metrics, investor behavior, and interactive dashboard reporting.

---

## Objectives

* Perform ETL on mutual fund datasets
* Clean and validate raw data
* Analyze fund performance and risk
* Calculate Sharpe Ratio, Sortino Ratio, Alpha, Beta, CAGR, VaR, CVaR, and Maximum Drawdown
* Study investor transaction patterns
* Build an interactive Power BI dashboard
* Generate actionable business insights

---

## Dataset Sources

1. Fund Master
2. NAV History
3. AUM by Fund House
4. Monthly SIP Inflows
5. Category Inflows
6. Industry Folio Count
7. Scheme Performance
8. Investor Transactions
9. Portfolio Holdings
10. Benchmark Indices
